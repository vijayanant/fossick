ಬಾಳೆಂಬ ಹೆಮ್ಮರದಲಿ \
ಚಿಗುರೊಡೆದಿದೆ ಹೊಸ ಆಸೆ \
ಮನವೆಂಬ ಆಗಸದಲಿ \
ಹೊಸದೊಂದು ಸೂರ್ಯೋದಯ 

ಮೊಟ್ಟೆಯೊಡೆದ ಮರಿಗೆ \
ಹೊಸತನದ ಬೆಳಕು \
ಹಸಿದ ಹಕ್ಕಿಯ ಬಾಯ್ಗೆ \
ಗುಟುಕಿನ ಕಾಳು 

ಹಕ್ಕಿಗಳು ಹಾಡುತಲಿವೆ \
ಸಾಂತ್ವನದ ಹಾಡು \
ಪುಟ್ಟ ಜೀವಕೆ ಕೊಟ್ಟಿವೆ \
ಆರೈಕೆಯ ಸೂರು 

ಆಡುತಿವೆ ಹಕ್ಕಿಗಳು \
ಒಂದಾಗಿ ಸೇರಿ \
ಕಾಮನ ಬಿಲ್ಲೊಂದು \
ಕಾಣುತಿದೆ ಅಲ್ಲಿ 

ಹಾಡುತಲಿದೆ ಕೋಗಿಲೆಯು \
ಮಾವಿನ ಚಿಗುರುಂಡು \
ಗರಿಬಿಚ್ಚಿದೆ ನವಿಲು \
ಸಂವೃದ್ಧಿಯಾ ಕಂಡು

ಜೀವವು ಬಂದಿದೆ ಜೀವನಕೆ \
ಹೊಸತನ ಬಂದಿದೆ ಆಲೋಚನೆಗೆ
